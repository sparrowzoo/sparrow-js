# 初始化nextjs 项目

```shell
npx create-next-app@latest
```

# 初始化shadcn

```shell
cd react-next-admin
npx shadcn@latest init
```

# 初始化common 库

```shell
yarn add cpy
```

// package.json

```
    "copy": "cpy ./../common/src/common  ./src/ --parents"
```

# 安装组件

```shell
yarn shadcn add input //最新版本
npx shadcn-ui@latest add table //
```
