What is Babel?

Babel is a JavaScript compiler Babel is a toolchain that is mainly used to convert ECMAScript 2015+ code into a backwards compatible version of JavaScript in current and older browsers or environments.

Using Babel

https://babeljs.io/setup#installation

https://www.jsdelivr.com/

将cdn.jsdelivr.net域名替换为fastly.jsdelivr.net或者gcore.jsdelivr.net

例如：https://cdn.jsdelivr.net/npm/tinymce-all-in-one@4.9.3/tinymce.min.js

替换为：

https://fastly.jsdelivr.net/npm/tinymce-all-in-one@4.9.3/tinymce.min.js

https://gcore.jsdelivr.net/npm/tinymce-all-in-one@4.9.3/tinymce.min.js
